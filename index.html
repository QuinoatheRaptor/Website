My Programming Portfolio

Hi there! My name is Joaquin Fernandez. Welcome to my portfolio of my proudest coding projects on github. I am currently a sophomore in college studying Computer Science at Benedictine College. My favorite projects to code are text-based games and simulations.


Every link below will take you to a new page with a link to the project's github as well as videos showing some of the most important features of each project. Feel free to look around and see all the cool things I've made!

Java Projects

Java is the first language I learned and the language I'm most comfortable with. All my most complex projects thus far have been done in Java.

Tic Tac Toe Program

A simple program that simulates a game of Tic Tac Toe. The user can play against one of three difficulties of bots or another user.

Racing Season Simulator

An in-depth program that simulates a full season of a racing league, keeping track of team and individual points.

C++ Projects

With my dream job being game design, I began learning C++ and trying to make some projects in it.

Infinite Coin Flipper

The Infinite Coin Flipper was my first personal C++ project, designed to run some experiments in probability. It simulates flipping a coin as many times as you want and tracks some statistics about the flips.

Monte Carlo War

Monte Carlo War was my first larger C++ project. It simulates a variation on the card game War that allows the players to bet a certain number of cards to win or lose during a war.
